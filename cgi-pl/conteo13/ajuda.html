Como configurar?
Para configurar edite as linhas do arquivo conteo.cgi indicadas abaixo:

     #!/usr/bin/perl
 Pergunte para seu provedor de hospedagem qual o path do PERL em seu servidor. Essa é a primeira linha do script.

     $pathsendmail = '/usr/sbin/sendmail';
Modifique o path do sendmail, programa de email do servidor.

     $pronto = "http://www.seusite.com.br/confirmado.html";
Edite para o endereço do seu site e crie um arquivo chamado confirmado.html que confirma que o amigo foi indicado. OBS:Crie esse arquivo de confirmação.    

     @urls = ('www.seusite.com.br','seusite.com.br');
Edite para o endereço do seu site.

     $paginaerro = "http://www.seusite.com.b/erro.html";
Edite para o endereço do seu site e crie um arquivo chamado erro.html que avisa que faltou dados a preencher no formulário ou foi preenchido indevidamente.

 

Parte texto do Conteo
Edite as duas linhas abaixo:

     $Subject = "nomedosite";
Troque o termo "nome do site" pelo nome do seu site.

     $site = "www.seusite.com.br";
 Edite para o link do seu site usando a mesma estrutura.

     $mensagem  e  $mensagem2
Recomendamos não alterar essas linhas.

Enviando para o servidor
Depois de tudo configurado, você deve passar os arquivos para o servidor. Vejamos então:
» Transfira o arquivo conteo.cgi para o seu diretório /cgi-bin/  e dê permissão 755 (chmod 755)
» Transfira o arquivo que contém o formulário, para qualquer diretório do seu site.

Como utilizar o Conteo 1.3 ?
 No arquivo form.html existe um formulário. Você deverá no código fonte mudar a linha "http://www.seusite.com.br/cgi-bin/conteo.cgi" de acordo com seu site.

Depois cole o formulário na página de seu site e pronto. 
OBS: para os que não sabem, o formulário está entre <form> e </form>
