--------------
ARQUIVOS
--------------

incluir.cgi   = Inclui mais sites para a busca
busca.cgi     = Script Primario
busca2.cgi    = Este script é acionado quando você usa o botão avançar
layout.htm    = O layout do site está neste arquivo, e você pode edita-lo
index.htm     = Página inicial do site de busca
queries.htm   = Este arquivo contém as palavras que foram usadas para pesquisar.
urls.db       = Contém o banco de dados de sites cadastrados.
config.txt    = Arquivo onde fica as configurações do BuscMaster


------------------
INSTALAÇÃO
------------------

 1: Arrume o caminho para o perl em todos os arquivos .cgi.

 2: Faça o Upload dos arquivos: layout.htm, busca.cgi, busca2.cgi, queries.htm, e urls.db para
        seu diretório CGI-BIN.

        NOTA: Usuários UNIX/LINUX, faça o upload dos arquivos .cgi em ASCII e
        de as permissões permições: 755 (chmod) e em queries.htm, config.txt e urls.db para 666.

 3: Arrume as váriaveis no arquivo config.txt

 4: Customize o arquivo LAYOUT.HTM.

Parabéns você terminou, Boa sorte!
