EZHomesite Documentation and Installation Instructions

Documentation:
This script actually originated as homepage.pl (Home Page Maker ) by David Palmer but in its initial form, it allowed only 3 choices for backgrounds and one single page to be created. Even then, it was an impressive program as it made page creation easy and fun - for anyone. Modifications were made by Greg Mathews webmaster@notts.net http://www.notts.net.

EZHomepage by Manny Juan
Then came the enhanced version EZHomepage by Manny Juan (manny@jps.net at http://inet-images.com/manny/). Manny Juan modified it by simplifying the setup (no more redundant makepage.html) and by allowing full editing of text and clip art choices. He also included a starter set of clip art images for splash graphics, backgrounds, horizontal separators, bullets and animated email icons so there are more choices available for creating a wide variety of pages.

EZHomepage 2000 by Bill Hall
First major upgrade of ezhp2000 was supplied by Bill Hall (bycs@bycs.com at http://http://www.bycs.com/ezhp2000/) August 8. 2000. He later added image upload capabilities, and fixed some security related bugs. In the same month he practically rewrote the script so it could handle multiple communities. In September 2000 he also added image management handling for admins. Bill Hall has made a lot of great improvements, and have also enhanced some of the other versions of EZHomepage.

Ethos Online's Version (by Simon Pierce)
Simon Pierce (simon@ethosonline.com at http://www.ethosonline.com/scripts/) also made a version of EZHomepage. He improved the interface and streamlined the script to allow easier control over headers/footers by owner and to allow a more friendly interface for the user including easier text entry, font selection, and even a help page.

Abuzer Khan's Version
Abuzer Khan's (san_trino@yahoo.com at http://www.hotscripts.com/detailed/7040.html) version builds on the mods made by Bill Hall and adds more login features: Login Id, Email-Id and Password. The login ID creates a new directory, and generates a index.html page in the directory. He also added a javascript that makes a window pop up everytime a page is opened, like on Geocities. This may be customized by the webmaster to be an ad, a banner, a reminder, etc.

EZHomesite Modifications Ver 1.00 - 01/10/2000
Modified by Per Kristian Johansen - pkj@internor.com

This is a modification of Abuzer Khan's (san_trino@yahoo.com) version and it hats been modified to create up to five pages to build a complete website

These modifications has been made:

- EZHomesite creates multiple pages. The generator now creates a 5-page Homesite.
- Added option to select contrasting background color.
- Customized navigation menu.
- Content and title different in each page.
- Option to center pages.
- Changed fontselection

Here's what the script can do:
Creates multiple pages (up to 5 total) for the website: (index, page2 thru page5) and all pages reside in a user directory
Allow selection of contrasting background color
Automatic building of navigation menu from page titles
Content, title, and style may be different in each page
Option to center page content
Allows font selection
Clip art selection

Installation Instructions
unzip Unzip the file ezhomepg.zip into your main webdirectory and maintain its directory structure. The unzipping wil create a directory 'ezhomesite'.
ezhomesite The ezhomesite subdirectory at your site should be writeable as this is where all user pages and reference files will be created.

Should be writeable as this is where all user pages and reference files will be created

images Upload the images directory (and contents) using ftp. 
ezhomesite
  images
    backgrounds
    bullets
    email
    lines
    pictures
note: the logo, header.jpg should be placed in /ezhomesite/images

Remove any extraneous files possibly left behind by ftp programs (ws_ftp usually leaves behind a log file called ws_ftp.log). If they are not deleted, they are included as clip art selections (but broken)!

index.html Edit and change the link to the logo image (images/header.gif) and the links to the cgi script.
Upload this to /ezhomesite
allow read and write permissions (chmod to 777)
data.txt Upload this to /ezhomesite
Allow read and write permissions (chmod to 777)
ezhomesite.pl Edit this file and adjust the parameters to agree with the website directory structure
Upload this to your cgi directory
Allow read and execute permissions (chmod to 755)
